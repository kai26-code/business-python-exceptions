# business-python-exceptions

Exercise 1: Handling invalid sales input
# Validates sales input, prevents invalid or negative entries & calculates total revenue. 

Exercise 2: Inventory quantity checker
# Validates inventory inputs with exceptions. 
# Also alerts when inventory falls below the reorder threshold. 

Exercise 3: Customer age verification 
# Validates customer age using exceptions.
# Determines eligibility for age restricted promotions and handles both ValueError and NameError.    

Exercise 4: Financial ratio calculator
# Calculates profit margin.
# Demonstrates defensive programming, handling ValueError and ZeroDivisionError.    

Exercise 5: Employee salary adjustment 
# Simulates salary adjustments & prompts user to enter an employee's current salary and an adjustment percentage. 
# Once valid inputs given, calculates and displays the employee's new adjusted salary.
